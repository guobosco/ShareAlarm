# UI 优化实施计划

## 1. 交互与易用性升级 (Usability)
- **扩大触摸热区**: 将 `HomeTopBar` 右上角菜单按钮尺寸从 `32.dp` 调整为 **48.dp**，符合人机交互标准。
- **优化空状态**: 在 `EmptyStateView` 中添加“新建提醒”引导按钮，提升转化率。

## 2. 视觉层级与美学重构 (Aesthetics)
- **卡片风格现代化**: 移除 `EventCard` 的 0.5dp 边框，改用更柔和的阴影或纯色块区分，减少视觉噪点。
- **强化文字层级**: 将事件标题字重调整为 `SemiBold`，字号微调至 `18.sp`，拉开与辅助信息的差距。
- **优化过期提醒**: 重新设计过期事件的折叠头部，使其视觉更轻量且不突兀。

## 3. 工程规范与一致性 (Consistency)
- **消除硬编码颜色**: 提取 `HomeScreen.kt` 中散落的 Hex 颜色值（如 `0xFF8E8E93`）到主题系统，确保深色模式（Dark Mode）完美适配。
- **字体优化**: 尝试将默认字体替换为 `FontFamily.SansSerif` 或系统圆体（如果可用），使界面更柔和。

## 4. 动效增强 (Animation)
- **列表进场动画**: 为时间轴列表项添加 `Modifier.animateItemPlacement` 或简单的淡入位移动画，提升流畅感。
