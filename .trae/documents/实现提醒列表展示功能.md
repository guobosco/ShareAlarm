# 实现提醒列表展示功能

## 问题分析
用户在创建提醒后，主页没有任何变化，没有显示创建的提醒内容。

## 解决方案

### 1. 修改 HomeScreen.kt
- 添加 `ReminderViewModel` 实例
- 添加提醒列表状态变量
- 在用户登录后加载提醒列表
- 替换示例提醒卡片为实际的提醒卡片列表
- 实现提醒卡片组件

### 2. 实现提醒时间排序
- 在 `HomeScreen` 中对提醒列表按事件时间排序
- 区分过期提醒和未过期提醒

### 3. 实现过期提醒折叠功能
- 为过期提醒添加折叠/展开状态
- 实现点击展开/折叠的交互逻辑

### 4. 确保数据同步
- 确保创建提醒后，主页能自动更新提醒列表
- 实现提醒列表的实时刷新

## 具体实现步骤

1. **修改 HomeScreen.kt**
   - 导入必要的依赖
   - 添加 `ReminderViewModel` 实例
   - 添加 `reminders` 状态变量
   - 在 `LaunchedEffect` 中加载提醒列表
   - 实现 `ReminderCard` 组件
   - 替换示例卡片为动态提醒列表

2. **实现提醒排序逻辑**
   - 在 `HomeScreen` 中添加排序逻辑，按事件时间升序排列
   - 区分过期提醒和未过期提醒

3. **实现过期提醒折叠功能**
   - 为 `ReminderCard` 添加展开/折叠状态
   - 为过期提醒默认设置为折叠状态
   - 实现点击展开/折叠的交互

4. **测试验证**
   - 创建多个提醒，验证排序功能
   - 创建过期提醒，验证折叠功能
   - 验证提醒创建后主页是否自动更新

## 技术要点
- 使用 `collectAsState` 监听提醒列表变化
- 使用 `LaunchedEffect` 在适当的时机加载数据
- 使用 `LazyColumn` 高效展示提醒列表
- 使用 `Card` 组件实现提醒卡片
- 使用 `mutableStateOf` 管理展开/折叠状态
